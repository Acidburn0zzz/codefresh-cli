Codefresh's cli is available for installation through NPM registry.

Install:
`npm install -g codefresh`

Install a specific version:
`npm install -g codefresh@{VERSION}`

Update to latest version:
`npm install -g codefresh`

Update to a specific version:
`npm install -g codefresh@${VERSION}`
