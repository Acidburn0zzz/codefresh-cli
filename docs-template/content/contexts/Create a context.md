### Command
`codefresh create context [type] [name]`

Create a context
### Positionals

Option | Default | Description
--------- | ----------- | -----------
name |  | Name of context
type |  | Type of the context
### Options

Option | Default | Description
--------- | ----------- | -----------
--owner | account | Owner of the context
--variable |  | Variables list
