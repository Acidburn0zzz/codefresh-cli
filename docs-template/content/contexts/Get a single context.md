### Command
`codefresh contexts [name]`

Get a specific or an array of contexts
### Positionals

Option | Default | Description
--------- | ----------- | -----------
name |  | context name
### Options

Option | Default | Description
--------- | ----------- | -----------
--type |  | Specific type of context
--owner |  | Owner of the context
