### Command
`codefresh create-context [name]`

Create or update an authentication context
### Positionals

Option | Default | Description
--------- | ----------- | -----------
name | default | Context name
### Options

Option | Default | Description
--------- | ----------- | -----------
--url | https://g.codefresh.io | Codefresh system custom url
--api-key |  | API key
